## ğŸ§ª Performance Considerations

This project uses several optimization techniques:

- **Mipmaps:** Enabled for all texture assets  
- **Static Batching:** Hub environment marked static  
- **GPU Instancing:** Applied to repeating materials (planets, props)  
- **LOD Groups:** Used for large 3D models for distance optimization  
- **Texture Atlases:** Some imported assets already include atlases  
- **IL2CPP Build Target:** Used for optimized builds  
- **Occlusion Culling:** Considered only where necessary  

Screenshots of optimization settings included in GDD.
